# Pastebin-Lite

A Pastebin-like web application built with Next.js and Postgres(Neon).

## Features
- Create and share text pastes
- Optional time-based expiry
- Optional view limits

## Tech Stack
- Next.js (React + Node.js)
- Postgres (Neon)

## Running Locally
1. npm install
2. Set DATABASE_URL in environment
3. npm run dev

## Persistence
Uses Postgres (Neon) to persist pastes across requests.
